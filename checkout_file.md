## git checkout [file]

**git checkout *[файл]*** - возвращает файл.

Если вы передумали оставлять внесённые в файл изменения, используйте команду:

```bash=
git checkout [файл]
```

Файл вернётся в состояние, в котором он находился во время последнего коммита, только учтите, что все изменения в этом файле исчезнут.
