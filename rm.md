## git rm

**git rm** - удаляет файл.

Чтобы удалить отслеживаемый неизменённый файл и проиндексировать это изменение, используйте команду:

```bash=
git rm [файл]
```

Чтобы удалить отслеживаемый изменённый файл и проиндексировать это изменение, используйте команду:

```bash=
git rm -f [файл]
```
